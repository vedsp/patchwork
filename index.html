git diff knows lines. Patchwork knows functions.

A semantic diff tool that understands your code — not just your characters. Built with tree-sitter.

📋 pip install patchwork-diff
View on GitHub
Works with Python & JavaScript · Git-native · TUI included

git diff shows you this

@@ -12,4 +12,6 @@ - const x = 10; - return x * 2; + const y = 20; + return y + 5;
→

Patchwork shows you this

~ processData (modified) + logger (added) - validateUser (deleted)
⬡

Function-level diffs

Understands what a function is. Shows you only what changed inside it.

⌥

Git-native

Works with commit hashes, branch names, HEAD~1. No file extraction needed.

◈

Tree-sitter powered

Built on the same parser used by Neovim, GitHub, and VS Code.

▣

TUI included

Side-by-side split view in your terminal. Syntax highlighted, keyboard driven.

◎

Python & JS

Supports Python and JavaScript/TypeScript out of the box.

⊕

One install

pip install patchwork-diff and you're done. No config, no setup.

patchwork diff HEAD~1 HEAD app.py
